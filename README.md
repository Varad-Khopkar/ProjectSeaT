# Maritime Training Platform (Demo)

This is the frontend demo repository for the gamified Maritime Training Platform.

## Current Objective

Build one fully playable demo mission, **Mission 01 - Port State Control (PSC)**, demonstrating an interactive, gamified learning experience to replace passive slide presentations.

## Development Stack

- **React 19**
- **Vite 8**
- **TypeScript**
- **Tailwind CSS v4**
- **React Router v7**
- **Lucide React**

## Documentation

For project design, path configuration, and file structure explanation, see **[STRUCTURE.md](./STRUCTURE.md)**.

## Getting Started

### Installation

```bash
npm install
```

### Running Locally

To run the development server locally:

```bash
npm run dev
```

### Building for Production

To compile and verify the build:

```bash
npm run build
```
